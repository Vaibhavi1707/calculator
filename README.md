# Calculator

## Introduction
A minimal scientific calculator. The following functions have been implemented:
- Finding Square root
- Factorial
- Natural Logarithm
- Power function

## Project stack
The project uses the following stack: 
- Source Code: Java
- Build Tool: Maven
- Version Control: GitHub
- CI: Jenkins
- Containerization: Docker
- Configuration Management: Ansible
